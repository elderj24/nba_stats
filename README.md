This analysis explores how correlated advanced NBA metrics are with player salaries. 

The analysis leverages data from the 2024-2025 season. 

The environment.yml file contains all the necessary packages to run the files. 

This repo includes the following:
- nba.scraping.ipynb (scrapes the necessary data from basketball-reference.com)
- nba_stats_sum.ipynb (visualizations and regression analysis)
- nba_stats_sum.py (raw code for the analysis)
- csv files with the underlying data
